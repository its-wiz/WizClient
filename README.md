# WizClient
 An open source Minecraft Hacked Client (1.8.9 + Optifine).


Features:
- HUD Overlay
- Duel HUD
- Enabled Moudles Overlay
- Rainbow HUD :D
- Show FPS
- Keybinding
- Command System
- Discord Integration
- Set different mode for modules.

<br>
Modules:
<br>
- Fly 
<br>
- Sprint 
<br>
- Animations
<br>
- NoSlowdown
<br>
- Longjump
<br>
- Bunnyhop
